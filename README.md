作品2：数字展示课程--无界书店
课程要求：
设计一个处于图书馆中书店的运营的方案。

在图书馆中设立书店，使书店自然具有 “与图书馆联动”的功能 和 校园责任感 。
体现在以下方面：
与图书馆联动:
  1. 图书馆->书店：由一个App承担主要联动功能。图书馆主要功能是书籍借阅，如果学生 
  在图书馆看到一本喜欢的书，认为有收藏价值，则可以通过App扫描书上的二维码，进行 
  下单，最后在书店取书；
  2. 书店->图书馆：书店每月畅销书会上架图书馆，面向全体师生借阅，间接促进书店图书销售。

校园责任感方面：
  1. 邀请各领域出色人才面向校园进行讲座；
  2. 为学生提供创意交流场所，如新媒体展览场地，为学生提供创意交流场所，定期开放场地，供学生们举行如新媒体展览、服装设计展览的交流活动；
  3. 为在校生设置工作机会，如微信公众号运营，具有使校园联系社会的责任；
  4. 支持学生创意。如接受学生的文创设计投稿，投票第一名的设计由书店进行制作文创产 
 品。书店和设计者分割最终收益；
  5. 书店还可以定期发布一些图书的电子版，供师生阅读。

书店具体设计，如场地分布等，请下载作品查看。
